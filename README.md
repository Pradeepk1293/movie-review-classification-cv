# Movie Review Classification Using Computer Vision

## Description
This project analyzes movie reviews using text sentiment analysis, OCR-based text extraction from images, and facial emotion detection.

## Features
- Text review sentiment analysis
- OCR text extraction from images
- Facial emotion detection
- Final positive/negative sentiment prediction
- Flask web interface

## Technologies Used
- Python
- Flask
- Scikit-learn
- TensorFlow
- OpenCV
- Tesseract OCR
- DeepFace

## Project Structure
- app.py
- templates/index.html
- static/project.css
- sentiment_model.pkl
- vectorizer.pkl

## How to Run
1. Install required libraries
2. Run `python app.py`
3. Open `http://127.0.0.1:5000`

## Output
The system predicts whether the movie review is Positive or Negative using text and visual analysis.
